## Strategic Notes

### Why 3 People?

- **Credibility:** A team signals organizational capability, not a solo consultant
- **Specialization:** Tech Leader brings depth Fermín can't deliver alone (automation builds, system integration)
- **Scalability:** Fermín stays at strategic level while Agustín and Tech Leader handle execution
- **Capacity:** Fermín at ~7–8 hrs/week can run Taylor alongside other engagements

### INTAKE as Standalone

- At 40–45% margin, INTAKE is profitable as a standalone diagnostic
- Even if Taylor doesn't proceed to DISCOVERY, TransformAZ walks away with $7.3K–$8K gross profit
- The Transformation Roadmap and Implementation Proposal (in DISCOVERY) position the next sale

### DISCOVERY Margin Pressure

- At 29–36% margin, DISCOVERY is tighter — the 3-person onsite trips drive this
- **Key lever:** Tech Leader rate. At $70/hr → 36% margin. At $100/hr → 29% margin
- If margin needs to improve: reduce Tech Leader onsite presence to 1 trip instead of 2 (saves ~14 hrs / $980–$1,400)

### Land-and-Expand Path

```
INTAKE ($18K) → DISCOVERY ($24K incremental) → IMPLEMENTATION ($80K–$150K) → RETAINER ($4K–8K/month)
                                                 ↑ This is where the real revenue is
```
